# Senior Project

## Setup

Install all dependencies: 
```
pip install -r requirements.txt
```

Setup MySQL database instance: 
```
cd database_files
```
```
python3 setup.py
```

To execute (Windows): 
```
python main.py
```

To execute (Mac/Linux): 
```
python3 main.py
```
